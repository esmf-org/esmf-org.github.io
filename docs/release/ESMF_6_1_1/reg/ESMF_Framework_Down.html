DOWNLOAD THE EARTH SYSTEM MODELING FRAMEWORK ESMF_6_1_1 VERSION

This is release ESMF_6_1_1 of the ESMF software. We are providing a tar.gz file of the ESMF source code.

README files are included. For additional build documentation see the ESMF User Guide, (html) for general documentation see the ESMF Reference Manual for Fortran (html) and the ESMF Reference Manual for C (html).

ESMF Source


Note that downloading the source ".tar.gz" file with Windows Internet Explorer may result in the filename suffix being changed to ".tar.tar". Please rename the file by changing the extra ".tar" suffix back to ".gz" before attempting to unzip it. Similarly, downloading with Firefox on Windows may result in the filename suffix being changed to ".tar.gz.tar". Before attempting to unzip it, please rename the file by deleting the extra ".tar" suffix, back to ".tar.gz". If you have any other difficulties downloading, installing or running the ESMF, please contact esmf_support@ucar.edu.
